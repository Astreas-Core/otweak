## 🚀 What is Origin Tweaks?
Origin Tweaks is a powerful tool designed specifically for OriginOS (vivo/iQOO) users. By utilizing **Shizuku**, the app can inject system-level configurations and unlock hidden UI customization features—all **without requiring root access**.

## 🌟 Key Features

### 🎨 Visual & UI Enhancements
* **Realtime Glow Effect:** Force-enable the premium gaussian blur and glow effects on budget OriginOS devices that normally have them disabled.
* **Transparent Widgets:** Make your launcher widgets fully transparent for a minimal, clean homescreen aesthetic.

### ⚙️ System Tweaks
* **Disable Force Rotate Icon:** Get rid of the annoying rotation suggestion icon that pops up in the corner when tilting your device.

### 🕒 Lock Screen Personalization
Take absolute control over your lock screen with direct property injections:
* Customize primary and secondary clock colors using precise HEX values.
* Adjust the transparency and alpha of lock screen elements.
* Change the default clock font style and layout structure.

### 📱 Premium App Experience
* **Material You Theming:** Beautiful Dynamic Colors that adapt to your wallpaper.
* **Pure Black Night Mode:** Battery-saving AMOLED dark mode.
* **In-App Updates:** Check for the latest releases directly from GitHub within the app's settings.
* **Seamless Shizuku Integration:** Easy onboarding to get your permissions sorted out quickly.

## ⚠️ Requirements
* An Android device running **OriginOS**.
* **[Shizuku](https://shizuku.rikka.app/)** must be installed and running (via Wireless Debugging or ADB).
* Android 12+.

## 📥 Installation
1. Go to the [Releases](../../releases) page.
2. Download the latest `app-release.apk`.
3. Install the APK on your device.
4. Launch the app and follow the on-screen instructions to grant Shizuku permissions.
